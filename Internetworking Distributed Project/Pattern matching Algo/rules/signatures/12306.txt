Rule:

--
Sid:
12306

--
Summary:
This event is generated when network traffic that indicates  Microsoft Messenger is being used.

--
Impact:
Possible policy violation. The use of  Microsoft Messenger may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the  Microsoft Messenger is being used on the protected network.

--
Affected Systems:
All systems using  Microsoft Messenger

--
Attack Scenarios:
This is a possible policy violation, it may be that  Microsoft Messenger has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of  Microsoft Messenger on the protected network and enforce or implement an organization wide policy on the use of  Microsoft Messenger.
