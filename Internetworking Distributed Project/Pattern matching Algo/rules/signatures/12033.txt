Rule:

--
Sid:
12033

--
Summary:
This event is generated when network traffic that indicates Jabber is being used.

--
Impact:
Possible policy violation. The use of Jabber may be prohibited by corporate policy in some network environments. 

This event indicates that the content of a transaction using Jabber has been replaced.

--
Detailed Information:
This event indicates that the Jabber is being used on the protected network.

--
Affected Systems:
All systems using Jabber

--
Attack Scenarios:
This is a possible policy violation, it may be that Jabber has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Jabber on the protected network and enforce or implement an organization wide policy on the use of Jabber.
