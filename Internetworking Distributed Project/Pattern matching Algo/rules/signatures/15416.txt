Rule:

--
Sid:
15416

--
Summary:
This event is generated when network traffic that indicates ICQ is being used and that the transaction has been negated.

--
Impact:
Possible policy violation. The use of ICQ may be prohibited by corporate policy in some network environments. 

This event indicates that the content of a transaction using ICQ has been replaced.

--
Detailed Information:
This event indicates that the ICQ is being used on the protected network.

--
Affected Systems:
All systems using ICQ

--
Attack Scenarios:
This is a possible policy violation, it may be that ICQ has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of ICQ on the protected network and enforce or implement an organization wide policy on the use of ICQ.
