Summary:
This event is generated when network traffic that indicates a Microsoft Word file with an embedded Flash file has been detected in network traffic.

--
Impact:
Unknown.

--
Detailed Information:
This event indicates that network traffic indicating that a Microsoft Word file with an embedded Flash file has been detected.

--
Affected Systems:
All systems.

--
Attack Scenarios:
Attackers may exploit systems by embedding certain filetypes within other files.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Ensure that all systems and applications are kept up-to-date and are patched with the highest patch level available from the vendor.
