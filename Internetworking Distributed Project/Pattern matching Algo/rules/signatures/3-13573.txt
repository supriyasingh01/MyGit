3-13573

Summary:
This event is generated when an attempt is made to exploit a known vulnerability in Office.

--
Impact:
Denial of Service. Information disclosure. Loss of integrity. 

--
Detailed Information:
Unspecified vulnerability in Microsoft Outlook in Office 2000 SP3, XP SP3, 2003 SP2 and Sp3, and Office System allows user-assisted remote attackers to execute arbitrary code via a crafted mailto URI.

--
Affected Systems:
Microsoft Office 5

--
Attack Scenarios:


--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Upgrade to the latest non-affected version of the software.

Apply the appropriate vendor supplied patches.
