Rule

--
Sid
17341

--
Summary:
This event is generated when shellcode is detected in network traffic.

--
Impact:
Unknown.

--
Detailed Information:
This event indicates that shellcode has been detected in network traffic. In particular, this event is generated when x86 OS agnostic alpha UTF8 tolower avoidance decoder is detected.

--
Affected Systems:
All systems.

--
Attack Scenarios:
An attacker may supply shellcode in the payload of an exploit attempt against a host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Check the destination host for signs of compromise.
