Sid
23284

--
Summary:
This event is generated when an attempt is made to exploit a known vulnerability in fusion middleware.

--
Impact:
Denial of Service. Information disclosure. Loss of integrity. 

--
Detailed Information:
Unspecified vulnerability in the Oracle WebCenter Forms Recognition component in Oracle Fusion Middleware 10.1.3.5 allows remote attackers to affect confidentiality, integrity, and availability via unknown vectors related to Designer.

--
Affected Systems:
oracle fusion middleware 10.1.3.5

--
Attack Scenarios:
An attacker can host a web site that will return code of their choosing to the object when a victim clicks on a malicious link.

--
Ease of Attack:
Medium.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Upgrade to the latest non-affected version of the software.

Apply the appropriate vendor supplied patches.
