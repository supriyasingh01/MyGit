Summary:
This event is generated when an attempt is made to exploit a known vulnerability in windows.

--
Impact:
Denial of Service.   

--
Detailed Information:
Unspecified vulnerability in the remote procedure call (RPC) component in Microsoft Windows XP SP2, XP Professional x64 Edition, Server 2003 SP1 and SP2, Server 2003 x64 Edition and x64 Edition SP2, and Vista and Vista x64 Edition allows remote attackers to cause a denial of service (RPCSS service stop and system restart) via a crafted RPC NTLMSSP authentication request.  NOTE: this also affects Windows 2000 SP4, although the impact is an information leak.
