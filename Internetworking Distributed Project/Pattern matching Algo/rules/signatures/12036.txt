Rule:

--
Sid:
12036

--
Summary:
This event is generated when network traffic that indicates IRC is being used.

--
Impact:
Possible policy violation. The use of IRC may be prohibited by corporate policy in some network environments. 

This event indicates that the content of a transaction using IRC has been replaced.

--
Detailed Information:
This event indicates that the IRC is being used on the protected network.

--
Affected Systems:
All systems using IRC

--
Attack Scenarios:
This is a possible policy violation, it may be that IRC has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of IRC on the protected network and enforce or implement an organization wide policy on the use of IRC.
