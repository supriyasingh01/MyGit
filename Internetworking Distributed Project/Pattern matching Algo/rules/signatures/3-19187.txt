Sid
3-19187

--
Summary:
This event is generated when an attempt is made to exploit a known vulnerability in forefront threat management gateway.

--
Impact:
Denial of Service. Information disclosure. Loss of integrity. 

--
Detailed Information:
The NSPLookupServiceNext function in the client in Microsoft Forefront Threat Management Gateway (TMG) 2010 allows remote attackers to execute arbitrary code via vectors involving unspecified requests, aka "TMG Firewall Client Memory Corruption Vulnerability."

--
Affected Systems:
microsoft forefront threat management gateway 2010

--
Attack Scenarios:


--
Ease of Attack:
Simple. Exploits exist.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Upgrade to the latest non-affected version of the software.

Apply the appropriate vendor supplied patches.
