Rule:

--
Sid:
12032

--
Summary:
This event is generated when network traffic that indicates MSN is being used.

--
Impact:
Possible policy violation. The use of MSN may be prohibited by corporate policy in some network environments. 

This event indicates that the content of a transaction using MSN has been replaced.

--
Detailed Information:
This event indicates that the MSN is being used on the protected network.

--
Affected Systems:
All systems using MSN

--
Attack Scenarios:
This is a possible policy violation, it may be that MSN has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of MSN on the protected network and enforce or implement an organization wide policy on the use of MSN.
