Rule:

--
Sid:
15561

--
Summary:
This event is generated when network traffic that indicates AOL Aimexpress is being used.

--
Impact:
Possible policy violation. The use of AOL Aimexpress may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the AOL Aimexpress is being used on the protected network. This application is a chat client and as such, it's use may violate corporate policy.

--
Affected Systems:
All systems using AOL Aimexpress

--
Attack Scenarios:
This is a possible policy violation, it may be that AOL Aimexpress has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of AOL Aimexpress on the protected network and enforce or implement an organization wide policy on the use of AOL Aimexpress.
