Sid
13292

--
Summary:
This event is generated when an attempt is made to exploit a known vulnerability in Skype.

--
Impact:
Denial of Service. Information disclosure. Loss of integrity. 

--
Detailed Information:
Unspecified vulnerability in the skype4com URI handler in Skype before 3.6 GOLD allows remote attackers to execute arbitrary code via "short string values" that result in heap corruption.

--
Affected Systems:
Skype Technologies Skype 18

--
Attack Scenarios:


--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Upgrade to the latest non-affected version of the software.

Apply the appropriate vendor supplied patches.
