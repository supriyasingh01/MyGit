Rule:

--
Sid:
16443

--
Summary:
This event is generated when network traffic that indicates Gmail chat is being used.

--
Impact:
Possible policy violation. The use of Gmail chat may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the Gmail chat is being used on the protected network. This application is a chat client and as such, it's use may violate corporate policy.

--
Affected Systems:
All systems using Gmail chat

--
Attack Scenarios:
This is a possible policy violation, it may be that Gmail chat has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Gmail chat on the protected network and enforce or implement an organization wide policy on the use of Gmail chat.
