Rule:

--
Sid:
12182

--
Summary:
This event is generated when network traffic that indicates Adobe FLV file transfer is being used.

--
Impact:
Possible policy violation. The use of Adobe FLV file transfer may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the Adobe FLV file transfer is being used on the protected network.

--
Affected Systems:
All systems using Adobe FLV file transfer

--
Attack Scenarios:
This is a possible policy violation, it may be that Adobe FLV file transfer has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Adobe FLV file transfer on the protected network and enforce or implement an organization wide policy on the use of Adobe FLV file transfer.
