Rule:

--
Sid:
15570

--
Summary:
This event is generated when network traffic that indicates Google Talk is being used and that the transaction has been negated.

--
Impact:
Possible policy violation. The use of Google Talk may be prohibited by corporate policy in some network environments. 

This event indicates that the content of a transaction using Google Talk has been replaced.

--
Detailed Information:
This event indicates that the Google Talk is being used on the protected network.

--
Affected Systems:
All systems using Google Talk

--
Attack Scenarios:
This is a possible policy violation, it may be that Google Talk has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Google Talk on the protected network and enforce or implement an organization wide policy on the use of Google Talk.
