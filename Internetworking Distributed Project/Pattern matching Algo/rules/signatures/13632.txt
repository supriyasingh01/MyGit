Rule:

--
Sid:
13632

--
Summary:
This event is generated when network traffic that indicates Zango adware is being used.

--
Impact:
Possible policy violation. The use of Zango adware may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the Zango adware is being used on the protected network.

--
Affected Systems:
All systems using Zango adware

--
Attack Scenarios:
This is a possible policy violation, it may be that Zango adware has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Zango adware on the protected network and enforce or implement an organization wide policy on the use of Zango adware.
