Rule:

--
Sid:
12611

--
Summary:
This event is generated when network traffic that indicates ebuddy.com is being used.

--
Impact:
Possible policy violation. The use of ebuddy.com may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the ebuddy.com is being used on the protected network.

--
Affected Systems:
All systems using ebuddy.com

--
Attack Scenarios:
This is a possible policy violation, it may be that ebuddy.com has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of ebuddy.com on the protected network and enforce or implement an organization wide policy on the use of ebuddy.com.
