Rule:

--
Sid:
3-7019

--
Summary:
This event is generated when network traffic that indicates the peer-to-peer application Winny is in use.

--
Impact:
Possible policy violation. The use of Winny may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the peer-to-peer application Winny is being used on the protected network.

--
Affected Systems:
All systems using Winny

--
Attack Scenarios:
This is a possible policy violation, it may be that Winny has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of Winny on the protected network and enforce or implement an organization wide policy on the use of [% msg %].
