Rule

--
Sid
3-15299

--
Summary:
This event is generated when an attempt is made to exploit a known vulnerability in Microsoft Visio.

--
Impact:
Unknown.

--
Detailed Information:
Microsoft Office Visio 2002 SP2, 2003 SP3, and 2007 SP1 does not properly perform memory copy operations for object data, which allows remote attackers to execute arbitrary code via a crafted Visio document, aka "Memory Corruption Vulnerability."

--
Affected Systems:
Microsoft Visio 2002
Microsoft Visio 2003 

--
Attack Scenarios:
An attacker needs to supply a specially crafted Visio document containing the right conditions for exploitation along with code to be executed on the target system.

--
Ease of Attack:
Difficult.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Upgrade to the latest non-affected version of the software.

Apply the appropriate vendor supplied patches.
