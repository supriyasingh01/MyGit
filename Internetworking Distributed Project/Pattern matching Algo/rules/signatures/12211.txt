Rule:

--
Sid:
12211

--
Summary:
This event is generated when network traffic that indicates P2PTv TVAnts is being used.

--
Impact:
Possible policy violation. The use of P2PTv TVAnts may be prohibited by corporate policy in some network environments. 

--
Detailed Information:
This event indicates that the P2PTv TVAnts is being used on the protected network.

--
Affected Systems:
All systems using P2PTv TVAnts

--
Attack Scenarios:
This is a possible policy violation, it may be that P2PTv TVAnts has been installated on a client host.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Disallow the use of P2PTv TVAnts on the protected network and enforce or implement an organization wide policy on the use of P2PTv TVAnts.
