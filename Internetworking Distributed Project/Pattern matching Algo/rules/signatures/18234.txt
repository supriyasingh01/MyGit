Rule:

Sid:
18234

--
Summary:
This event is generated when a QuickDraw/PICT file was detected in a download.

--
Impact:
Serious. Possible malformed file leading to exploitation of a program that parses this file type.

--
Detailed Information:
The download of a QuickDraw/PICT file was detected.

--
Affected Systems:

--
Attack Scenarios:
An attacker may influence a client to download a malicious file that is then used to compromise a machine through software that process this file type.

--
Ease of Attack:
Simple.

--
False Positives:
None known.

--
False Negatives:
None known.

--
Corrective Action:
Ensure the system is using an up to date version of the software and has had all vendor supplied patches applied.
